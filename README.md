# Smart Home Automation using IoT

## Description
This project demonstrates an IoT-based home automation system
using ESP8266 and Node.js.

## Technologies Used
- ESP8266 (NodeMCU)
- Node.js
- Express.js
- Wi-Fi Communication

## Features
- Remote ON/OFF control
- Real-time response
- Scalable system

## Applications
- Smart homes
- Building automation
- Metro station electrical monitoring
